== Intro

=== What's this?

This repo contains the code base for polster's blog based on link:https://jekyllrb.com/[Jekyll] and link:https://pages.github.com/[GH Pages].

=== Prerequisites

* link:https://jekyllrb.com/docs/installation/[Jekyll installation]
* link:https://jekyllrb.com/docs/continuous-integration/github-actions/[Jekyll GH Actions]

== Get started

* Clone this repo
* If not done already, install jekyll and bundler gems:
[source,shell]
----
gem install jekyll bundler
----
* Build the site locally and run it:
[source,shell]
----
bundle install
bundle exec jekyll build
bundle exec jekyll serve
----

== Hints

=== GitHub Actions

When publishing Jekyll sites using GH Pages as CDN, GH offers a built-in GH Actions workflow that can be configured.

Go to GH Repo >Settings > Pages for more details.
